3. Fabrication

Nested components laid out on a 4 foot by 8 foot sheet

CNC Milling

The hives have been designed to be cut using one standard size wood board of 1220mm x 2440mm or 4' x 8'. This drawing shows the geometry and cut strategies that a machine operator will need to use with their own CAM software.

To help with the creation of the machine 'tool paths', we have split the strategies into color coded layers and named them according to the strategy required. See the 'Nesting' and 'Cut Strategy' files.
