Sheet Thickness - 18mm/0.708"

File Name Type Description
3D Model .3dm Original 3D model made in Rhino CAD.
3D Strategies .STL 3D information needed to cut an angle using your CAM software.
Nesting .3dm Flat layout for cut strategies and tool paths.
STL Model .STL Provides an easy way to import into web viewers. Not for modeling.
Cut Strategies .DXF Import into CAM software. The layers contain the cut strategies.
Window Cuts .DXF Contains the cutting strategies to load into your CAM software to cut the window panes.
